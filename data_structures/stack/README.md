
# Stack (LIFO)

This folder contains a simple Python implementation of a Stack (Last-In, First-Out).

---

## 🧩 What Problem It Solves

Stacks let you process data in **reverse order** from how it was added.

Useful whenever the most recent item must be processed first.

---

## ⏱ Time & Space Complexity

| Operation | Time |
|----------|------|
| Push | O(1) |
| Pop | O(1) |
| Peek | O(1) |
| Space | O(n) |

---

## 🌍 Real-World Uses

- Browser history  
- Undo/Redo functionality  
- Call stack (function execution)  
- Depth‑first search (DFS)  
- Expression parsing  

---

## 📄 Implementation File

`stack.py` includes:

- pop()
