
 CS Foundations — Core Data Structures Implemented From Scratch

This repository contains hand-built implementations of fundamental computer science data structures:

- Dynamic Array
- Stack (LIFO)
- Queue (FIFO)
- Hash Map (with chaining)

These implementations are written from scratch to deepen understanding of how data is stored, accessed, and manipulated at a low level.

---

## 📚 Why This Project Exists

Most high-level languages hide the inner workings of data structures.  
But strong engineers understand what happens under the hood.

This project builds that foundation by implementing:

- Memory growth strategies  
- Index-based access  
- Amortized time complexity  
- Hashing and collisions  
- LIFO & FIFO behavior

## 🎯 Purpose of This Project

This project is focused on **building strong foundations** in computer science.

By implementing data structures manually, you learn:

- How memory is allocated  
- How indexing and hashing work  
- Why certain operations are O(1) or O(n)  
- What real systems use these structures  
- How Python’s built‑ins work internally  

This is a great starting point for:
- Algorithms  
- System design  
- Interview preparation  
- Backend engineering  
- General CS mastery
